Requisitos computacionais


A TerraMA² está disponível para download nas versões Windows, Linux e Mac. Para instalação completa da TerraMA2 outros programas devem ser instalados e configurados corretamente. Maiores detalhes da instalação e configuração estão no manual de instalação da plataforma.

Os sistemas operacionais testados foram:

      • Linux : Ubuntu 16.04
      • Windows 10
      • Mac


Importante: Detalhes da instalação veja “Manual de Instalação”.


As principais dependências são:

  • Servidor de banco de dados – deve-se instalar o PostgreSQL com a extensão espacial PostGIS (ambos gratuitos).
  • Aplicativo TerraView – programa do INPE utilizado para fazer acesso aos dados estáticos que serão utilizados por um projeto no módulo de administração. Até mesmo os dados dinâmicos poderão ser visualizados com os recursos desse aplicativo. Ferramentas de análise ainda poderão ser úteis para produzir novos dados a serem utilizados pela plataforma TerraMA2.
  • Servidor de mapas Geoserver – serviço de publicação dos mapas estáticos e dinâmicos que estarão disponíveis no banco de dados e serão publicados nos formatos WCS, WFS e WMS.
